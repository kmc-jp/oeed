oeed
====

Open Engineer Energy Drink.

材料(分量: 1l)
------

* 炭酸水 750ml 
* フレーバーシロップ(メロン) 250 ml 
* カフェイン(錠剤) 600mg 
* スッポン粉末 350mg
* エラブ粉末  350mg
* 赤マムシ粉末 350mg 
* トナカイの角粉末 350mg
* 高麗人参六年根粉末 350mg
* マカ粉末 350mg 
* ガラナ粉末 350mg

作り方
------

1. カフェイン(錠剤)をすり鉢等ですり潰す
2. 容器に、フレーバーシロップを入れて、炭酸水で4倍に希釈する
3. 2に、粉末状になったカフェインと各種粉末を入れる
4. よくかき混ぜる
